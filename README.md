# agents-company

AIエージェントお仕事ゲーム企画書
コンセプト
「プレイヤーは社長！マスコット風エージェントたちが奮闘するお仕事解決ゲーム」 ユーザー（社長）が依頼した仕事を、個性豊かなマスコットキャラクターたち（AIエージェント）が受け取り、一生懸命考えて解決していく姿を楽しむシミュレーションゲーム。

・使用技術
フロントエンド: React Native
バックエンド: Python / FastAPI
API: Claude API　OpenAI API　Deepseek API　Gemini API

・世界観とUI構成
秘密基地風の3つの部屋を行き来します。 社長室: 社長の拠点。仕事の依頼、完了報告の受け取り、新規エージェントの採用を行う。 会議室: エージェント達がタスクの解決方法を相談する場所。Claude APIによる自律的な問題解決プロセスの可視化。 作業室: 会議で決まった手順に従い、エージェント達が実際に作業する場所。

・ゲームサイクル
最初は社長1人の秘密基地からスタートします。簡単な仕事をこなし、報酬を獲得して新しいエージェントを雇用。エージェントが増えることで、より複雑な仕事を会議で分担してこなせるようになります。

・キャラクター設定
バックエンドエージェント: 熱血でマッチョなエンジニア。健康管理も完璧にこなし、「重いデータ処理は俺の筋肉とサーバーで支える！」というパワー系。

フロントエンドエージェント: ドジっ子。デザインセンスは抜群だが抜けている。「完璧なUIができました！あ、ボタンのリンク繋ぎ忘れた…」と作業室で慌てる。

料理系エージェント: 超真面目。几帳面で論理的。「必要なカロリーと栄養素の要件定義から入ります」と堅苦しく進める。
